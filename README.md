## 2. Differnce Between structured and unstructured data
Structured data is highly organized and fits neatly into databases (rows and columns), while unstructured data is messy, lacks a predefined format, and includes things like videos, emails, and social media posts. 
### Examples include
**Stuctured**
-Employee database – Organized tables with fields like employee ID, name, department, and salary.
-Bank transaction records – Each entry has a fixed format: transaction ID, date, amount, sender, receiver.
**Unstructured**
-Social media posts – Tweets, Facebook updates, or Instagram captions with text, images, hashtags, and emojis all mixed together.
-Customer emails – Free-form text, attachments, and varying formats that don’t fit neatly into rows and columns.

## 3.What is a Schema?
A schema is a mental framework that helps us organize and interpret information. For example, if you have a schema for "supermarket," you might expect certain things to be present, such as Dairy products, Beverages, Electronics, and Cosmetics.

## Difference between a list and a dictionary
**List**
A list is an ordered collection of elements. Each element can be of any type, and duplicates are allowed.
**Dictionary**
 A dictionary is an unordered collection of key-value pairs. Each key must be unique, but values can be duplicated.
